  • You can add a Section at the top asking for the respondent's department for better tracking
Step 2 — Download Responses from MS Forms
  1. Open your form at forms.office.com
  2. Click the Responses tab
  3. Click "Open in Excel" or "Download responses"
  4. Save the .xlsx file to your computer
The exported file has columns: ID, Start time, Completion time, Email, Name followed by your question columns. TMS automatically skips the first 5 columns.
Step 3 — Import to TMS

Upload the downloaded Excel file. TMS reads the question columns and imports all valid responses into TNI.

Only .xlsx / .xls files from Microsoft Forms.
TMS validates every row:
Employee code exists in {{ plant_name }} ({{ emp_count }} active employees)
Programme Name is not empty
Skips duplicate/empty rows
Failed rows → downloaded as error Excel
