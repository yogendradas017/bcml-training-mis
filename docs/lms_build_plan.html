Out of Scope — Phase 1
📱
Mobile App
Responsive web works on phone; native app = 3x dev time. Revisit if adoption proven.
🎬
Video Hosting / Streaming
Render disk not suited for video files. SCORM can embed Vimeo/YouTube inside the package.
🔑
SSO / SAML
BCML has no SAML IdP. Emp_code + generated password is simpler and works now.
💳
Billing / Multi-Tenant SaaS
Phase 2 if proven at BCML and decision to sell globally. Needs PostgreSQL + Stripe.
📧
Email Notifications
BCML has no SMTP config. Can add later with SendGrid / Resend (5 min task once ready).
🔵
Blue Collar Employees
No email/device infrastructure. LMS Phase 1 targets WC only. BC training stays in TMS.
